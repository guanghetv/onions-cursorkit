# CursorKit

此项目用于维护洋葱私有插件市场。

## 新增插件

给到cursor "参考/docs/add-a-plugin.md" 新增插件 `X插件` ;

## 安装使用

Cursor Setting -> Plugins -> onions-plugins -> 进入插件市场 -> 选择插件 - >点击 "Add to Cursor"

**建议前端安装 Common + Frontend**
**建议后端安装 Common + Backend**

## 现有插件
- Backend(后端开发插件)
    - rules 规范
    - skills 技能
    - command 命令
- Frontend（前端开发插件）
    - rules
    - skills
    - command

- Common 通用
